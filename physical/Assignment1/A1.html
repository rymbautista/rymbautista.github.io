            void setup() {           // initialize digital pin LED_BUILTIN as an output.
                pinMode(13, OUTPUT); // initialize digital pin 13 as an output
                pinMode(12, OUTPUT); // initialize digital pin 12 as an output
                pinMode(11, OUTPUT); // initialize digital pin 11 as an output
              }
              
            void loop() {            // loop function that runs forever
            digitalWrite(13, HIGH);  // turns on the LED in Pin 13 the (first one)
            delay(1000);             // wait for a second
            digitalWrite(13, LOW);   // turns off the LED in Pin 13 the (first one)
            digitalWrite(12, HIGH);  // turns on the LED in Pin 12 the (second one)
            delay(1000);             // wait for a second
            digitalWrite(12,LOW);    // turns off the LED in Pin 12 the (second one)
            digitalWrite(11, HIGH);  // turns on the LED in Pin 11 the (third one)
            delay(1000);             // wait for a second
            digitalWrite(11, LOW);   // turns off the LED in Pin 11 the (third one)
            delay(1000);             // wait for a second
            }
Code that initializes the use of pins 11,12, and 13 on the Arduino and then loops through a cycle that sets a pin to High (on) or low (off) starting with turning on the first LED for one second and then it turns off while the second pin turns on then off while the third LED turns on for a second and then off. The pattern is then repeated.
